# IS-4M

¿Qué es? Es un algoritmo que encuentra la forma de conectar todos los nodos de un grafo gastando lo menos posible mediante una estrategia voraz.

¿Para qué sirve? Si lo vemos como dinero o material: Sirve para unir varios puntos entre sí (como una red) gastando la menor cantidad posible de recursos.

¿Cómo se implementa en el mundo? Se utiliza para diseñar redes de infraestructura. Por ejemplo, se usa para conectar ciudades a la red eléctrica o de fibra óptica usando la menor cantidad de cable posible.

¿Cómo lo implementarías en tu vida? Lo usaría para planificar la instalación de las mangueras de un sistema de riego en mi jardín, buscando gastar la menor cantidad de tubo para conectar todas las plantas.

¿Cómo lo implementarías en tu trabajo o tu trabajo de ensueño? Si tuviera una empresa de telecomunicaciones, lo utilizaría para optimizar el tendido de cables de internet entre las oficinas de mis clientes, reduciendo al mínimo los costos de material.
